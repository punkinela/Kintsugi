## Avoiding Common Mistakes

### ‚ùå DON'T:
- Focus only on "wins" or "accomplishments"
- Make it seem like only successes matter
- Imply challenges are weaknesses to overcome and forget
- Use Kintsugi as mere decoration without explanation
- Separate "Own Your Impact" and "Kintsugi" as different ideas

### ‚úÖ DO:
- Always pair wins with resilience/growth language
- Explicitly state that challenges ADD value
- Connect the metaphor to practical impact
- Use "Own Your Impact + Kintsugi" as ONE unified philosophy
- Emphasize this is especially powerful for those facing barriers

## Quick Reference

When adding ANY new feature or updating ANY component, ask:
1. **Does it acknowledge BOTH wins and resilience?**
2. **Does it connect to the Kintsugi philosophy when appropriate?**
3. **Does it make clear that challenges ADD value?**
4. **Would someone facing barriers feel their full impact is recognized?**

If the answer to any is "no," update the messaging to include the unified philosophy.
